파이썬 패키지 제작 사용자 안내서¶

파이썬 패키지 제작 사용자 가이드 에 오신걸 환영한다. 본 페이지는 모던한 도구들로 파이썬 패키지들을 배포하고 설치하는 튜토리얼과 래퍼런스들을 제공한다.

GitHub 에 Python Packaging Authority 의해 관리된다. 우리는 언제나 피드백을 환영한다 contributions and feedback. 😊

시작하기¶

파이썬 개발 생태계에서 작동하는 핵심적인 도구들과 개념들을 알아보려면 Tutorials절을 참조한다:

  • 패키지를 설치하는 방법에 대해 배우려면 tutorial on installing packages참조한다.
  • 버전 관리되는 프로젝트의 의존성들을 관리하려는 것을 배울려면 doc:tutorial on managing application dependencies <tutorials/managing-dependencies>을 참조한다.
  • 패키지들과 프로젝트들을 배포하려면 tutorial on packaging and distributing을 참조한다.
  • 파이썬 라이브러리들과 애플리케이션의 패키징 옵션에 대해 살펴보려면 Overview of Python Packaging참조

더 알아보기¶

Tutorials절 넘어 가이드란엔 아래와 같은 내용들을 소개한다.

  • Guides절은 Installing pip/setuptools/wheel with Linux Package Managers이나 Packaging binary extensions과 같은 내용을 다룬다.
  • Discussions절은 doc:discussions/deploying-python-applications이나 pip vs easy_install과 같은 주제에 대해 깊이있는 래퍼런스를 다룬다.
  • PyPA specifications절은 패키지 상호운용성(interoperability) 스펙에 대해 다룬다.

추가적으로, other projects에는 파이썬 패키지 위원회 멤버들이 관리하는 목록들이 있다.
