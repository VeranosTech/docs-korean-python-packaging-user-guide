Python Packaging User Guide¶

Python Packaging User Guide 에 온것을 환영한다. 이는 최신 도구를 이용하여 Python package를 배포하고 설치하는데 도움이 되는 튜토리얼과 참고문헌의 모음이다.

이 가이드는 GitHub 상에서 Python Packaging Authority 에 의해 관리되고 있다.

Note

만약 legacy PyPI에서 pypi.org로의 migration하는데 도움이 필요하다면

Migrating to PyPI.org 를 참조한다.

Get started¶

Python packaging ecosystem과 작업에 필요한 필수 도구와 개념은 Tutorials 부분에서 커버한다:

  • package를 설치하는 법을 배우려면 tutorial on installing packages 를 참조.
  • project의 package를 만들고 배포하는 법을 배우려면 tutorial on packaging and distributing 을 참조

Learn more¶

Tutorials 외에, 이 가이드는 몇가지를 더 포함한다:

  • Installing pip/setuptools/wheel with Linux Package Managers 또는 Packaging binary extensions 과 같은 Guides 섹션의 따라하기 도움말
  • Deploying Python applications 또는 pip vs easy_install 과 같은 Discussions 섹션의 주제에 대한 심층적 참고문헌
  • packaging interoperability specification을 위한 PyPA Specifications 섹션

추가적으로 Python Packaging Authority에 의해 관리되는 other projects 목록이 있다.
